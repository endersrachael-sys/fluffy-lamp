🤖 How It Works
JarDIYn uses an agentic loop — Claude reads your question and autonomously decides which tools to call (or none at all). No hardcoded routing. The agent may call:

identify_plant — diagnose symptoms
get_garden_zone — USDA zone & frost
get_soil_data — SoilGrids (ISRIC)
get_weather_forecast — Open-Meteo/NOAA
get_frost_alerts — NOAA NWS alerts
get_pollen_forecast — pollen levels
get_historical_weather — recent rain
generate_diy_report — monthly report
lookup_plant_database — OpenFarm + iNaturalist

Tool calls appear below each response.
